Digit classificator

This is a digit classificator using deep learning. The algorithm is based on the book Make your own neural network by Tariq Rashid, but i wrote it in javascript, and added suppport for multi layer networks and the tanh function. The dataset used to train this network was a part of the MNIST dataset of handwrited numbers The matrix library is from a youtube tutorial about neural networks made by a channel called the coding train. You can check the full code of this webpage in my github repo HERE

How to use it? : you draw a number (only one at a time can be classified) and then press the get result button to ask the neural network what number it thinks it is, and the answer will appear below, to start again press clear canvas...

You can also train the network: draw a number, select the answer in the right side of the page and press the train button, have in mind that training takes time, so you probably have to draw a lot of numbers...The learning depends on the training, you can train this network to classify only one number for example, so it's up to you...(the training is lost if you reload the page)
